## The Algorithm

1. **Определите user need**: Что пользователь/бизнес реально хочет — anchor карты сверху.
2. **Разложите на components**: Всё, что нужно для need: apps, data, infra, integrations.
3. **Расставьте по evolution**: Genesis (новое) → Custom → Product → Commodity (utility).
4. **Нарисуйте dependencies**: Кто от кого зависит — снизу вверх по value chain.
5. **Найдите inefficiency**: Custom build для commodity; missing platform for repeated genesis.

## Anti-patterns

- **Map всей компании за раз**: Начните с одного value stream.
- **Статичная карта**: Evolution движется — пересматривайте quarterly.
- **Map без action**: Карта для презентации, без strategic moves — waste.
